# Triage labels

The engineering skills use five canonical triage roles. This repository uses the default label names.

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | More information is required from the requester |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an implementation agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation or external authority |
| `wontfix` | `wontfix` | Intentionally not planned |

All label commands must explicitly target `2233admin/opencli-admin`.
